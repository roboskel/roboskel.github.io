Color Post

May 18, 2019

What a colorful post!

This is an idea that came from xukimseven/HardCandy-Jekyll looking at this cheerful and colorful them, I wanted to enable something similar for mine.

You can go fork and star hers too! üòâ

How does it work?

Basically you need to add just one thing, the color:

---
layout: post
title: Color Post
color: brown
---

It can either be a html color like brown (which look like red to me). Or with the rgb:

---
layout: post
title: Color Post
color: rgb(165,42,42)
---

The background used is lineart.png from xukimseven you can edit it in the config file. If you want another one, put it in /assets/img as well.
